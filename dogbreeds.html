Data Cleaning in SQL

üéØ Goal and Research Questions

The goal of this project was to clean, transform, and analyze a dog breeds dataset using Microsoft SQL Server. The primary research questions included:

  • Which countries have the largest number of dog breeds?
  • What are the most common health problems among dog breeds?
  • What is the average longevity of dog breeds?
  • What is the average height of dog breeds?

üìä Data Sources

The dataset used is a dog breeds dataset from Kaggle, stored in Microsoft SQL Server under the table PortfolioProject.dbo.DogBreeds. It includes columns such as breed name, country of origin, common health problems, longevity, and height.

üîÑ Process (Cleaning, Transformation, Exploration)

  • Imported the dataset into SQL Server and performed initial exploration using SELECT * and COUNT
  • Identified top countries by breed count using GROUP BY Country_of_Origin and ORDER BY
  • Split the Common_Health_Problems column into three separate columns (Health_Problem_1, Health_Problem_2, Health_Problem_3) using PARSENAME after replacing commas with periods
  • Trimmed spaces and standardized naming (e.g., converted health problem names to lowercase)
  • Unified synonymous health problems (e.g., "Eye issues" ‚Üí "Eye problems")
  • Dropped the original Common_Health_Problems column to reduce redundancy
  • Calculated the most common health problems by combining all three new columns using UNION ALL and GROUP BY
  • Transformed longevity and height ranges into numerical averages for further analysis
  • Performed exploratory analysis to understand distributions and trends in breed characteristics

üìà Results / Findings

  • Identified the countries with the largest number of dog breeds, highlighting global breed diversity
  • Determined the most common health problems across all breeds
  • Calculated average longevity and height for breeds, providing insights into typical breed characteristics
  • Cleaned and transformed the dataset to make it analysis-ready, enabling easier querying and aggregation

‚ö†Ô∏è Limitations and Future Work

  • The dataset contains only breed-level data, so individual variation is not captured
  • Some health issues may have been oversimplified when splitting and standardizing health problem columns
  • Ranges for longevity and height are averaged, which may lose some variability
  • Future work could integrate additional datasets (e.g., behavioral traits, exercise needs, diet) for deeper insights
  • Advanced analysis, such as clustering breeds by health risk or predicting breed longevity based on multiple features, could be explored
